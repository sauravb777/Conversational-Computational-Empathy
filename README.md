## Overview

In modern customer-centric systems, traditional call support often lacks emotional understanding and fraud awareness. This project addresses that gap using Transformer-based deep learning models to analyze and respond to customer interactions effectively.

The system processes transcribed call data and performs:

- Intent Classification  
- Sentiment Analysis  
- Fraud Detection  

Based on this understanding, it generates context-aware and empathetic responses.

---

## System Pipeline

1. **Speech-to-Text Conversion**  
   Customer call audio is transcribed into text.

2. **Text Processing**  
   Cleaned and prepared for model input.

3. **Transformer-based Analysis**  
   - Intent detection  
   - Sentiment evaluation  
   - Fraud identification  

4. **Response Generation**  
   Generates empathetic and context-aware replies.

5. **Text-to-Speech Output**  
   Converts response back to audio with a natural delay (~2 seconds pause threshold).
